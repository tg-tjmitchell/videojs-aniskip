videojs-aniskip Example

This is a sample page that demonstrates how the videojs-aniskip plugin works.

Plugin Options

For this demo, the following options are configured:

{
  skipRanges: [
    { start: 5, end: 10, type: 'intro' },
    { start: 20, end: 25, type: 'outro' }
  ],
  buttonText: 'Skip Segment',
  notificationText: 'Skipping segment...'
}

This will show a skip button when the video reaches second 5 or second 20.
